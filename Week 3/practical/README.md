#  Week 3 – Practical Application

##  Overview
This folder contains the practical implementation work for Week 3 of Vulnerability Assessment and Penetration Testing (VAPT).

The practical section focuses on applying theoretical concepts in controlled lab environments using tools such as Metasploit, Burp Suite, sqlmap, OWASP ZAP, Wireshark, and OpenVAS.

##  Practical Tasks Covered

### 1. Advanced Exploitation Lab
- Exploit chaining
- PoC customization
- Exploit result documentation

### 2. Web Application Testing Lab
- DVWA testing
- SQL Injection testing
- XSS testing
- Burp Suite request interception

### 3. Reporting Practice
- Findings table
- Remediation plan
- Stakeholder briefing

### 4. Post-Exploitation and Evidence Collection
- Session logging
- Traffic capture
- Evidence hashing
- Chain-of-custody documentation

### 5. Capstone Project
- Full VAPT cycle
- Vulnerability detection
- Exploitation
- Remediation and reporting

##  Disclaimer
All practical work must be performed only in authorized lab environments such as DVWA, Metasploitable, VulnHub, or TryHackMe.
